# Plataforma Kitanda - Busca e Comparação de Produtos

## Descrição
Plataforma web para busca e comparação de produtos de diferentes fornecedores, desenvolvida para pequenas e médias empresas.

## Funcionalidades
- Sistema de busca inteligente de produtos
- Comparação de produtos lado a lado
- Filtros avançados (preço, fornecedor, categoria)
- Catálogo de fornecedores verificados
- Interface responsiva e acessível
- Sistema de usuários (compradores e fornecedores)

## Tecnologias Utilizadas
- HTML5 semântico
- CSS3 com variáveis e Flexbox/Grid
- JavaScript vanilla (ES6+)
- Design responsivo
- Padrões W3C
